# character-creation-rest-api

## Description
Yep a simple Fake REST-API for testing, The data I use are just random things that pop out of my mind.

## Usage

```bash
# Install dependencies
pnpm

# Development
pnpm watch

pnpm dev
# or
pnpm ts-start

# Production
pnpm build

pnpm start
```
